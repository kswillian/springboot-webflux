# springboot-webflux

Exemplo de implementação de uma CRUD reativo com Spring WebFlux e MongoDB, utilizando as melhores praticas de desenvovimento e testes (Unitário e Integração).

## Tecnologias

* JDK 17
* Gradlew
* Spring WebFlux
* MongoDB Atlas
* Lombok
* MapStruct
* Jakarta validation
* JUnit5
* Anotações Personalizadas
* ReactiveMongoTemplate
* Records
